# 🧩 BlockPY

## A python framework that uses the ttkbootstrap library as a base for an easier development process.

## Working similarly with React's components only needing a ``ui.py`` file inside a component folder and an import inside ``main.py``.

# ✨ Pros

- ## Better and cleaner code with ttkbootstrap.

- ## Reusable widgets that work like React's components.

# Notes:

- ## You should learn at least a little bit of either ttkbootstrap, tkinter or both.

- ## Since this framework uses ttkbootstrap as a base for the components.

# ⚙ Getting Started

- ## Just download this repo's zip, extract it to a folder and run ``installDependencies.bat``, it'll install ``pyinstaller`` and then turn ``main.py`` into an executable file.
